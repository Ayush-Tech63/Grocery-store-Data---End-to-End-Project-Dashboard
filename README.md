Swiggy Instamart Sales Dashboard (Power BI Project)
Project Overview

This project presents an interactive Power BI dashboard built using a Swiggy Instamart dataset. The dashboard provides insights into sales performance, customer behavior, order trends, and category-wise performance.

The goal of this project is to demonstrate data visualization, dashboard design, and business insights extraction using Power BI.

Dashboard Preview

Key Insights
KPI Metrics
Total Sales: 1.20M
Total Orders: 8.523K
Average Rating: 3.92 ⭐
Average Item Weight: 12.86
Sales Insights
Highest sales come from Tier 3 cities (472K).
Tier 1 cities show comparatively lower total sales.
Sales trend shows a decline in 2024 compared to 2021–2023.
Category Performance

Top performing categories:

Fruits & Vegetables – 178.15K
Snack Foods – 175.43K
Household – 135.98K
Frozen Foods – 118.58K
Dairy – 101.28K
Shop Type Performance
Supermarket Type1 generated the highest revenue.
Average sales across shop sizes remain consistent (~141).
